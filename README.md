# AutoHub

Running DB:
- Start docker containers: `docker-compose up -d`
- From terminal, run `docker exec -it postgres_db bash`
- To connect to the PostgresSQL database, run `psql -U postgres -d AutoHub`
- Helpful commands:
  - List all tables: `\dt`
  - Describe a table: `\d <table_name>`
  - Quit shell: `\q`
